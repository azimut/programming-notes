- drives melody

    - with voltage divider to limit INPUT to 5v

** DONE 3-Mixer + Distortion
*** mixer
**** ~passive mixer~ circuit

  - can be used to mix N inputs
  - alone is not possible to drive anything
  - high output impedance
  - with equal resistors, ouputs the avg of both inputs

**** ~active mixer~ circuit

  - we add an OPAMP BUFFER, after the passive mixer
  - input signal goes to non-inverting input
  - =TL072=
  - adding a 1k resistor at its OUTPUT
    - to avoid shortcircuits, due OPAMP low impedance

**** ~inverting mixer~ circuit

- based on an OPAMP (as an inverting buffer) and a passive mixer
- input going to INVERTING INPUT
- with a 100k resistor on the INVERTING INPUT loop
- with NON-INVERTING INPUT to GND

**** ~double negative mixer~ circuit

- starting from the ~inverting mixer~
- while being inverted doesn't matter for audio
- it does for CV
- we add an inverter opamp circuit at the end to restore it as an OUTPUT
- we could keep an inverted OUTPUT

**** ~adjustable mixer~ circuit

just adds 100k potentiometer before each 100k INPUT RESISTOR
