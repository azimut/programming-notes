- Interfaces
  - Serial bus
    - Interfaces: SPI, I2C, Microwire, UNI/O, 1-Wire
    - Protocol:
      - Phases
        1) op-code: 8 bits
        2) address: 8/24 bits
        3) data: r/w
      - op-codes
        - WRite ENAbLe/DIsable
        - ReaD/WRite Status Register
        - READ/WRITE data
  - Paralel bus
    - faster
    - less popular
