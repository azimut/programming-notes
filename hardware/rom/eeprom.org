#+TITLE: EEPROM (72)

https://en.wikipedia.org/wiki/EEPROM
- Serial Electrically Erasable PROMs
- Interfaces
  - Serial bus
    - Interfaces: SPI, I2C, Microwire, UNI/O, 1-Wire
    - Protocol:
      - Phases
        1) op-code: 8 bits
        2) address: 8/24 bits
        3) data: r/w
      - op-codes
        - WRite ENAbLe/DIsable
        - ReaD/WRite Status Register
        - READ/WRITE data
  - Paralel bus
    - faster
    - less popular
- https://en.wikipedia.org/wiki/EEPROM
  - 1972 - first fabricated?
  - 1993 - first MCU with it

* Serial bus devices

- common interfaces: spi, i2c, microwire, uni/o, 1-wire

- protocol
  - phases
    1) op-code: usually the first 8bits input to the serial input pin of the device
       |--------+-----------------------|
       | WRENAL | Write enable          |
       | WRDI   | Write disable         |
       | RDSR   | Read status register  |
       | WRSR   | Write status register |
       | READ   | Read data             |
       | WRITE  | Write data            |
       |--------+-----------------------|
    2) address: 8-24 bits
    3) data

** Example: i2c write - arduino

#+begin_src arduino
  Wire.beginTransmission(device_address);
  Wire.write(word_address);
  Wire.write(data);
  Wire.endTransmission();
#+end_src
