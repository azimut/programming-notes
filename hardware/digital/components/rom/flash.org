- https://en.wikipedia.org/wiki/Flash_memory
  - vs EEPROM
    - programmed in blocks instead of bytes
    - needs explicit full erase
      - can only 1->0
      - or reset back to 1
  - Types
    - 1987 NOR flash architecture
      - SOIC8
      - allows single word writes
      - byte per byte rw
      - SPI interface
      - 25L
    - 1989 NAND flash architecture
      - TSOP48
      - read by blocks
      - accessed raw by the OS (no build-in controller)
      - eg: memory card, usb flash drive, ssd (2009-)
    - ???? eMMC Flash
      - BGA{153}
      - build-in controller
