* default operations/words
T = take
P = put
|------+---+---+----------------------|
|      | T | P |                      |
|------+---+---+----------------------|
| swap | - | - | top 2 elements       |
| dup  | - | 1 |                      |
| drop | 1 | - | trashes top of stack |
| emit | 1 | - |                      |
| rot  | - | - | rotates whole stack  |
| .    | 1 | - | pop                  |
| .s   | - | - | shows whole stack    |
|------+---+---+----------------------|
* snippets
|----------------+-------+-------------------------------------|
| .s             |       | show stack                          |
| 0 10 .         |     0 | pop top of stack                    |
| 10 20          | 10 20 |                                     |
| 10 20 +        |    30 |                                     |
| : add10 10 + ; |       | defines a new word "add10"          |
| 5 add10        |    15 |                                     |
| 2 dup          |   2 2 |                                     |
| 1 2 swap       |   2 1 |                                     |
| 10 emit        |       | prints ascii of the number on stack |
|----------------+-------+-------------------------------------|
